# Password Strength Checker

A simple Password Strength Checker built using HTML, CSS, and JavaScript. The application determines password strength based on the number of characters entered by the user.

## Features
- Real-time password checking
- Strength indication based on password length
- Weak, Medium, and Strong password levels
- Simple and user-friendly interface
- Instant feedback

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How It Works
- Less than 6 characters → Weak
- 6 to 10 characters → Medium
- More than 10 characters → Strong
